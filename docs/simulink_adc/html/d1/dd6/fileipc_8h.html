◆ IPC_flags

enum IPC_flags
Элементы перечислений
IPC_CREATE_FILE 

Всегда создает файл

IPC_OPEN_FILE 

Открывает файл (если не существует, то создается)

IPC_FILE_RDONLY 

Открывает файл в режиме только для чтения (если не существует, то ошибка)

IPC_FILE_WRONLY 

Открывает файл в режиме только для записи (если не существует, то создается)

IPC_FILE_RDWR 

Открывает файл в режиме записи/чтения (если не существует, то создается)

IPC_FILE_DIRECT 

Открывает файл в режиме прямого записи/чтения (Linux)
