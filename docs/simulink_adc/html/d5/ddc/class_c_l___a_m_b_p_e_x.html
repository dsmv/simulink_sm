Detailed Description

Работа с аппаратным модулем через библиотеку BARDY.

Описание библиотеки приведено в документе "DCR0328 - Управляющая библиотека BARDY".

Обращение к регистрам производится через службу REG0, в файле brd.ini требуется подключение devreg.

Класс обеспечивает инициализацию модуля, обращения к регистрам тетрады, работу со стримами.

Выделение памяти для стрима в режиме 8.

В режиме 8 массив ppBlk содержит физические адреса блоков памяти. Бит 0 адреса используется для факта создания прерывания. Наличие 1 означает что прерывание будет сформировано.
