SM_ADC Documentation

C++ S-function для приёма данных от АЦП через разделяемую память

Входные порты

  • START_IN - вход сигнала старта
  • GET_DATA - получение данных

Выходные порты

  • DATA - выход данных, массив 16384 отсчёта, double
  • START_OUT - выход сигнала старта

По фронту START_IN происходит передача через разделяемую память сигнала на начало сбора данных, после подтверждения начала сбора данных производится формирование фронта на выходе START_OUT

По фронту GET_DATA из разделяемой памяти принимается блок данных и передаётся на выход GET_DATA

Основные функции и классы:

  • BaseBlock - базовый класс для создания S-Function из библиотеки easyLink
  • Block - класс S-Function sm_adc
  • Block::start() - начало сеанса моделирования
  • Block::outputs() - формирование выходных сигналов
  • TF_SimulinkShared - обмен с приложением через разделяемую память
  • TL_SharedMemoryWrapper - управление разделяемой памятью

Для реализации S-Function используется библиотека easyLink-3.3.0
