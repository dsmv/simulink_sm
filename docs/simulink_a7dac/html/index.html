SIMULINK_A7DAC  1.0
Control of A7_DAC device
Программа управления генератором сигналов A7_DAC

Основные функции и классы:

  • BRDC_main() - точка входа
  • TF_SimulinkA7Dac - основной класс теста
  • TF_SimulinkShared - обмен с Simulink через разделяемую память
  • TL_SharedMemoryWrapper - управление разделяемой памятью
  • TF_A7DacCtrl - управление модулем A7_DAC
  • TF_PLL - управление синтезатором частоты на модуле A7_DAC
  • TF_DDS - управление выводом сигнала через ЦАП на модуле A7_DAC

Назначение параметров S-Function sm_ctrl

  • D0 - канал ЦАП 0 - частота сигнала
  • D1 - канал ЦАП 0 - начальная фаза сигнала
  • D2 - канал ЦАП 0 - амплитуда сигнала
  • D3 - канал ЦАП 1 - частота сигнала
  • D4 - канал ЦАП 1 - начальная фаза сигнала
  • D5 - канал ЦАП 1 - амплитуда сигнала
  • D6 - длина импульса в отсчётах тактовой частоты
  • D7 - резерв
